# 步骤2：统计分析

## 2.1 数据分析流程

### 2.1.1 数据预处理
1. **数据导入**：读取原始数据文件
2. **数据清洗**：
   - 缺失值处理
   - 异常值处理
   - 数据格式转换
3. **数据整合**：合并多个数据源

### 2.1.2 描述性统计
- 连续变量：均值±标准差或中位数(四分位数)
- 分类变量：频数(百分比)
- 图表展示：直方图、箱线图、条形图

### 2.1.3  Inferential Statistics
1. **组间比较**：
   - t检验（两组连续变量）
   - ANOVA（多组连续变量）
   - 卡方检验（分类变量）
2. **相关性分析**：
   - Pearson相关（连续变量）
   - Spearman相关（有序分类）
3. **回归分析**：
   - 线性回归
   - Logistic回归
   - Cox比例风险模型

### 2.1.4 统计软件要求
- 主要软件：Python (pandas, scipy, statsmodels)
- 辅助软件：R (ggplot2, survival)
- 输出格式：CSV、Excel、JSON

## 2.2 结果报告规范

### 2.2.1 报告结构
```
├── 研究设计概述
├── 基线特征表
├── 主要终点分析
├── 次要终点分析
├── 亚组分析（如适用）
├── 安全性分析
└── 敏感性分析
```

### 2.2.2 统计结果报告格式
1. **P值报告**：
   - p < 0.001：报告为 p < 0.001
   - 0.001 ≤ p < 0.05：报告精确值
   - p ≥ 0.05：报告为 p > 0.05

2. **置信区间**：
   - 格式：效应值 (95% CI, 下限-上限)
   - 例：OR = 1.5 (95% CI, 1.2-1.8)

### 2.2.3 图表规范
1. **表格**：
   - 使用三线表格式
   - 表格标题置于上方
   - 注释置于下方

2. **图形**：
   - 分辨率：≥300 dpi
   - 格式：PDF优先，其次PNG
   - 坐标轴标签清晰

### 2.3.2 结果验证
