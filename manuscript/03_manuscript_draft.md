# 步骤3：论文草稿写作

## 3.1 论文结构规范

### 3.1.1 标准SCI论文结构
```
1. Title（标题）
2. Abstract（摘要）
   - Background/Purpose
   - Methods
   - Results
   - Conclusions
3. Keywords（关键词）
4. Introduction（引言）
5. Methods（方法）
6. Results（结果）
7. Discussion（讨论）
8. Conclusion（结论）
9. Acknowledgements（致谢）
10. References（参考文献）
11. Tables（表格）
12. Figures（图表）
13. Supplementary Materials（补充材料）
```

### 3.1.2 各部分写作要求

#### 标题 (Title)
- 简洁明了（不超过15-20个单词）
- 包含主要研究因素和结局
- 避免使用缩写（除非广泛认可）

#### 摘要 (Abstract)
- 结构化摘要（IMRAD格式）
- 字数限制：通常250-300字
- 包含关键数据结果

#### 引言 (Introduction)
1. **研究背景**：介绍研究领域现状
2. **研究问题**：指出研究缺口
3. **研究目的**：明确研究目标和假设
4. **研究意义**：说明研究价值

#### 方法 (Methods)
- 详细描述研究设计
- 研究对象纳入/排除标准
- 干预措施描述
- 观察指标定义
- 统计分析方法

#### 结果 (Results)
- 客观呈现数据
- 使用图表支持
- 按逻辑顺序组织
- 包含关键统计结果

#### 讨论 (Discussion)
1. **结果总结**：概述主要发现
2. **对比分析**：与既往研究比较
3. **机制探讨**：解释可能机制
4. **局限性**：说明研究限制
5. **未来方向**：提出研究展望

## 3.2 写作规范

### 3.2.1 语言规范
- 英文：Times New Roman, 12号字
- 中文：宋体, 小四号字
- 行距：1.5倍
- 首行缩进：2字符

### 3.2.2 参考文献引用
- 格式：作者年份制（如：Zhang et al., 2023）
- 引用位置：句末或句中适当位置
- 优先引用高影响力文献

### 3.2.3 图表规范
1. **表格**：
   - 编号：Table 1, Table 2...
   - 标题：简洁描述内容
   - 注释：解释缩写和特殊符号

2. **图形**：
   - 编号：Figure 1, Figure 2...
   - 图注：包含必要说明
   - 分辨率：≥300 dpi
