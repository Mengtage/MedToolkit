# 如何启动 MedPaperHunter

## 最简单的方法（所有系统通用）

打开终端（Terminal / 命令提示符），进入项目文件夹，然后运行：

```bash
python3 start_web_app.py
```

---

## 按操作系统选择：

### Windows 🪟

1. 双击 `启动系统.bat`

### macOS 🍎

**方法 A（最简单）：**
1. 打开终端
2. 输入 `cd` 加空格，然后把 MedPaperHunter 文件夹拖进去
3. 按回车
4. 输入 `python3 start_web_app.py` 并回车

**方法 B：**
1. 右键点击 `启动系统.command`
2. 选择“打开”
3. 在确认对话框中再点“打开”

**遇到安全警告？** 查看 `docs/macOS_安全设置说明.md`

### Linux 🐧

1. 打开终端
2. 进入项目文件夹
3. 运行 `python3 start_web_app.py`

---

## 启动后：

系统会自动在浏览器打开 http://localhost:8000

如果浏览器没自动打开，手动访问这个地址即可！
