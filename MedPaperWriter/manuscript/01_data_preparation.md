# 步骤1：数据准备

## 1.1 参考文献收集

### 1.1.1 来源要求
- **PubMed**：主要检索来源
- **Embase**：补充检索
- **Cochrane Library**：系统评价检索

### 1.1.2 文献筛选标准
1. 纳入标准：
   - 研究类型：随机对照试验、队列研究、病例对照研究
   - 研究对象：人类受试者
   - 语言：中文或英文
   - 发表时间：近10年

2. 排除标准：
   - 重复文献
   - 会议摘要（无全文）
   - 综述类文献（除非作为背景引用）
   - 动物实验（除非特别说明）

### 1.1.3 文献数据结构
```json
{
    "pmid": "PubMed ID",
    "title": "文献标题",
    "authors": "作者列表",
    "journal": "期刊名称",
    "impact_factor": "影响因子",
    "pub_date": "发表日期",
    "abstract": "摘要内容",
    "doi": "DOI号",
    "keywords": ["关键词1", "关键词2"],
    "study_type": "研究类型",
    "sample_size": "样本量",
    "key_findings": "关键发现",
    "limitations": "局限性",
    "relevance_score": "相关性评分(1-5)"
}
```

## 1.2 研究方案整理

### 1.2.1 方案内容要求
- 研究背景
- 研究目的与假设
- 研究设计
- 研究对象（纳入/排除标准）
- 干预措施
- 观察指标（主要/次要终点）
- 样本量计算
- 统计分析方法

### 1.2.2 文件格式
- 支持格式：.txt, .md, .docx
- 编码：UTF-8

## 1.3 试验数据整理

### 1.3.1 数据来源
- 原始数据文件（Excel、CSV）
- 统计分析结果报告

### 1.3.2 数据质量控制
1. 数据完整性检查
2. 异常值检测
3. 缺失值处理说明
